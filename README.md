# Midpoint

### [결과 페이지]
> Kakao maps api를 활용하여 지도 상에 Midpoint를 확인할 수 있습니다.
> Polyline, Mapmarker를 활용해 보기에 편리한 UI를 구현하였습니다.
> Midpoint를 클릭하면 해당 장소의 링크로 연결됩니다.
> location1 -> Midpoint, location2 -> Midpoint로의 이동시간을 확인할 수 있습니다.
